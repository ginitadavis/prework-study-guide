HTML

  • The head element contains information about the webpage
  • The meta element contains information about the page that is used by the browser
  • The body element represents the visible content shown to the user
  • The footer usually contains the author, copyright, contact, sitemap, and navigation
  • Element and tag are used interchangeably. However, an element refers to the entire element from the opening to the closing tag, whereas a tag only refers to what appears inside the angle bracket

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Class atributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector
  • We can refer to color by its hexadecimal code or hex# which can be found here https://htmlcolorcodes.com

GIT

  • Git is a version control system that lets you manage and keep track of each version of your project
  • Git commands always start with git and we enter it in gitbash
  • We can use them to initiate or create a repo
  • Git is great for tracking changes locally on your own computer
  • git checkout -b branch-name: creates a new branch and switches to it
  • git status: checks what branch we are currently in
  • git add -A: Add changed files in our working directory
  • It's important to always sync the local version of the code with the remote version in GitHub
  • git commit -m "here goes what has been added": commit changes to our repository
  • git pull origin main: make sure our current branch is up to date before opening a pull request. Enter the following command
  • git push origin feature/here goes the branch name: Push up all the changes
  • Merge the PR so the main branch holds the updated code for the project
  • Open Pull Request (PR)

Javascript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • To link the information we added in javascript, we add a link element under the footer element in html. The style.css is linked within the head, the script.js is linked within the body element.
  • An array is a single variable that is used to hold a group of data, and tipically holds data that are related in some way
  • A function is a set of instructions that tells the computer how to perform certain tasks
  • for loop or conditional statements work automatically, but functions must be called in order to work
  • Open Chrome DevTools using Ctrl+Shift+I. We can debug using this tool directly
